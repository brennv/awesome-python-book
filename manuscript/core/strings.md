# Strings

```Python

>>> name = 'Fred'
>>> f"He said his name is {name}."
'He said his name is Fred.'

```


```Python

>>> import decimal
>>>
>>> width = 10
>>> precision = 4
>>> value = decimal.Decimal("12.34567")
>>>
>>> f"result: {value:{width}.{precision}}"  # nested fields
'result:      12.35'

```
